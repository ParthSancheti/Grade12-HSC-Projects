- **HTML**: `HTML_Files/SOP2/index.html`  
- **PHP**: `PHP_Files/SOP2/vowel.php`

- **HTML**: `HTML_Files/SOP3/index.html` (launcher)

- **PHP**: `PHP_Files/SOP4/marks.php`
